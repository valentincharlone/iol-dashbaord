InvertirOnLine API

v2REST⚠ Entorno REAL

Base URL: https://api.invertironline.com

Documentación completa de todos los endpoints disponibles en la API pública de InvertirOnLine. Todas las operaciones afectan el entorno real de inversión.

🔐 Autenticación

Todos los endpoints requieren un Bearer Token enviado en el header HTTP. El token tiene una validez de 15 minutos y se renueva con el refresh token.

Authorization: Bearer {tu_access_token}

Para activar el acceso: Mi Cuenta → Personalización → APIs
